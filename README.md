OpenGL simple app
* 3.3 core profile
* prereqs - conan, cmake
* deps - glfw, glew, imgui, glm
* run.cmd/run.sh

## How to build?

First time – `./build.sh`

Next time – `cmake -DCMAKE_BUILD_TYPE=Release --build build --target opengl-imgui-sample`

## How to run?

```shell script
cd build
./opengl-imgui-sample
```
